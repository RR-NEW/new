# ❤️ Love Calculator

A simple and interactive **Love Calculator** built using **HTML, CSS, and JavaScript**. Users can enter two names and receive a randomly generated love percentage along with a fun message.

## 📸 Preview

Enter your name and your partner's name, then click the **Calculate Love** button to discover your love score! ❤️

---

## 🚀 Features

- ❤️ Calculate a random love percentage
- 👤 Input fields for two names
- 🎨 Clean and attractive user interface
- 📱 Responsive design for desktop and mobile
- ⚡ Instant result using JavaScript
- 💖 Fun messages based on the generated score

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 How It Works

1. Enter your name.
2. Enter your partner's name.
3. Click the **Calculate Love** button.
4. The application generates a random love percentage.
5. A fun message is displayed based on the score.
